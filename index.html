Versionamento

O Git é um software que permite realizar versões de um projeto. O Github Desktop é um software que faz a ligação entre VS Code, Git e Github.

Github Desktop permite fazer a sincronia entre o repositório local e o repositório remoto.

Commit

É um processo de confirmação de uma versão entre vírgulas, ou seja, a cada interaçaõ é possível confirmar a alteração.

Push

Este comando envia (sincroniza) alterações realizadas no repositório local com um repositório remoto no Githib.
